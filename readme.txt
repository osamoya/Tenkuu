使い方：
1.dataフォルダに"tennkuuotosi_xxxx.bmp"という連番の画像を入れる
2."sketch_211009_Yukiri.pde"を起動する。
3.MAXに画像の枚数+1を代入する(ごめんなさい)
4.起動し、画像が読まれるのを待つ(3000枚だと約5分かかると予想されます)
5.画像が表示されたら、テボの位置をマウスでクリックする

操作：
マウスクリック：CSVで書き込む座標の指定
矢印キー　　　：次の画像や前の画像を表示する
エンターキー　：クリックで指定されたところまでを書きだして終了
